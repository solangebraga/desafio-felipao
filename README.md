Projeto desenvolvido em JavaScript para classificar o nível de um herói com base na quantidade de experiência (XP).
A solução utiliza variáveis para armazenar o nome do herói e o valor de XP, além de estruturas condicionais (if / else if / else) para determinar corretamente o nível correspondente.
Os intervalos de XP foram organizados em ordem decrescente, garantindo que não haja sobreposição ou lacunas na classificação. O nível "Ouro" foi ajustado para iniciar em 5001, mantendo a consistência dos intervalos.
Ao final da execução, o programa exibe uma mensagem informando o nome do herói e seu respectivo nível.
